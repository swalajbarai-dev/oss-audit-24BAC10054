## Scripts Description

### Script 1 — System Identity Report

Displays system information such as kernel version, user, uptime, and Linux distribution.

### Script 2 — FOSS Package Inspector

Checks whether a package is installed, shows version details, and prints a philosophy message.

### Script 3 — Disk and Permission Auditor

Audits system directories, showing permissions, ownership, and disk usage.

### Script 4 — Log File Analyzer

Reads a log file, counts keyword occurrences, and shows last matching lines.

### Script 5 — Open Source Manifesto Generator

Takes user input and generates a personalized manifesto saved in a text file.

---

## How to Run

Run in Linux (WSL):

```bash
bash script1.sh
bash script2.sh
bash script3.sh
bash script4.sh /var/log/dpkg.log error
bash script5.sh
```
